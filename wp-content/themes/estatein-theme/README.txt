== Description ==

Estatein is a custom WordPress theme designed specifically for real estate websites. It features:

* Clean, modern design focused on property listings
* Custom post types for Properties and Agents
* Responsive design that works on all devices
* Optimized for real estate business needs
* Removes unnecessary blog functionality
* Custom dashboard widgets for real estate management

== Installation ==

1. Upload the theme files to /wp-content/themes/estatein-theme/
2. Activate the theme through the WordPress admin panel
3. Go to Appearance > Customize to configure the theme
4. Add your properties using the Properties post type
5. Add your agents using the Agents post type

== Features ==
